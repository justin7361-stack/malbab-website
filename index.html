FEATURES

말밥AI의 핵심 기능

경마일 경주 분석부터 실시간 ELO 랭킹까지, 필요한 모든 것을 한 곳에.

🤖

AI 종합 분석

출전마 전체를 10가지 지표로 분석해 경주마 분석 지표를 산출합니다. 과거 수만 건의 경주 데이터로 학습된 머신러닝 모델이 실시간으로 분석을 제공합니다.

📊

10가지 과학적 지표

기본 능력, 최근 폼, 거리 적성, 주로 적성, 기수 케미, 조교사 폼, 컨디션, 밸류 지수, 구간 기록, 스크래핑 인사이트 — 10가지 지표를 점수화해 종합 평가합니다.

📡

ELO 강도 랭킹

경주 결과로 실시간 갱신되는 말별 동적 강도 지수. 최근 폼과 상대 전적을 반영합니다.

🏟️

주로 편향 분석

서울·부산경남·제주 각 경마장의 당일 주로 특성(내측·외측 유리도, 선행·추입 유리도)을 분석해 경주별 전술 예측을 제공합니다.

⚡

실시간 출마표 분석

KRA 출마표 공개 직후 자동으로 데이터를 수집·분석합니다. 경주 당일 아침 이미 모든 분석 결과가 준비됩니다.

📱

모바일 PWA

경마장 현장에서도 편리하게 — 스마트폰 브라우저에서 앱처럼 사용하는 PWA 방식으로 설치 없이 바로 이용 가능합니다.

🔍

스크래핑 심층 데이터

공식 API 외 경마 전문 미디어·커뮤니티 정보를 자동 수집·분석합니다. 조교 평가, 전문가 코멘트 등 현장 정보까지 더해 예측 완성도를 높입니다.

❓ 말밥AI는 마권 구매를 권유하나요?
아닙니다. 말밥AI는 데이터 기반 경마 분석 정보를 제공할 뿐, 마권 구매를 권유하거나 중개하지 않습니다. 최종 판단은 항상 이용자 본인이 합니다.
❓ 이용 연령 제한이 있나요?
유료 서비스는 만 19세 이상 성인만 이용 가능합니다. 미성년자의 유료 서비스 이용을 금지합니다.
❓ AI 분석 결과는 얼마나 정확한가요?
AI 분석 수치는 투명하게 공개합니다. 분석 결과는 참고 목적이며 실제 경주 결과와 다를 수 있습니다.
❓ 문의 및 고객지원은 어디서 받나요?
고객지원 이메일로 문의해주세요. 운영시간: 평일 10:00~18:00
⚠️ 이용자 안내 및 경고
말밥AI가 제공하는 모든 분석 정보는 참고 목적의 데이터 분석 결과이며 마권 구매 권유가 아닙니다. 경마 및 관련 도박은 중독될 수 있으며, 한국마사회 고객지원센터(1566-3333)에서 상담 받으실 수 있습니다.
